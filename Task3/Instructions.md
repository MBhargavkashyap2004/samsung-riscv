# Instructions and Opcodes
   **R type: Register type**
   **I type: Immediate type**
   **S type: Store type**
   **B type: Branch type**
   **U type: Upper immediate type**
   **J type: Jump type**

## Opcodes
Opcodes are the codes which determine the type of instructions

**R-type**: 0110011
**I-type**: 0010011
**S-type**: 0100011
**B-type**: 1100011
**U-type**: 0110111
**J-type**: 1101111
